# Interviews

요구사항을 명확히 하기 위한 **심층 인터뷰 산출물**을 보관하는 디렉토리입니다.

목표·범위·제약·완료 기준이 흐릿한 작업을 시작하기 전, `deep-interview` 스킬이 사용자와의 문답을 통해 의도를 정리하고 그 결과를 이 폴더에 기록합니다.

## 사용법

```
/deep-interview
```

- 요구사항 정리 후 → `superpowers:brainstorming`(설계 탐색) 단계로 이어집니다.
- 요청이 이미 구체적이거나 단순 설정 변경처럼 물을 가치가 낮은 작업에는 사용하지 않습니다.

## 파일명 규칙

```
YYYY-MM-DD-<topic>.md
```

## 워크플로 위치

이 폴더는 작업 워크플로 1단계 산출물입니다. 전체 단계는 [docs/README.md](../README.md)의 `작업 산출물`과 [CLAUDE.md](../../CLAUDE.md)의 `Workflow & Documents`를 참조합니다.
