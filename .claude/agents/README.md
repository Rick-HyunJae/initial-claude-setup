# Project Agents

이 프로젝트에서 사용 가능한 Claude Code Agent 목록.

| Agent                 | Description                                              |
| --------------------- | -------------------------------------------------------- |
| `compound-solutions`  | ce-\* 스킬의 단일 진입점 (write/read/maintain 모드 판별) |
| `plan-reviewer`       | plan-\* 스킬 라우팅 (키워드 기반 직무 선택)              |
| `code-reviewer`       | code-\* 스킬 라우팅 (키워드 기반 직무 선택)              |
| `session-compounder`  | ce-sessions 내부에서 dispatch되는 합성 전용 서브에이전트 |
